Browsers usually cannot launch .bat files directly for security. Use Obsidian Shell Commands, Commander/Buttons, or double-click the script from the file system.

Mac Path

Recommended for Jeremy/maintainers: double-click the Mac command file or run it through Obsidian Shell Commands.

scripts/build_printable_packets_mac.command
Open Build Artifacts Folder Open Setup Notes

If macOS blocks the command file, run this once in Terminal: chmod +x scripts/build_printable_packets_mac.command

iPad / Android Path

Mobile devices cannot run the Python builder directly from Obsidian. Use the already-built PDFs/DOCXs included in the vault, or print/save from a browser-friendly page.

Open Build Artifacts Folder

On iPad: open the PDF, tap Share, then Print or Save to Files. The printable packets should already exist in 08 Printable Study Materials/Build Artifacts.
