# EPA Approved Methods Index

> Operator-study index for EPA-approved drinking-water analytical method references. The CFR and incorporated method text control compliance use.

## Authority Order

```yaml
compliance_authority_order:
  - current CFR / incorporated approved method text
  - state primacy agency direction
  - exact approved method version/date
  - EPA approved-method summary table
  - certified laboratory SOP
  - manufacturer insert/manual
  - training reference
  - vault study note
```

## High-Yield Items

- RTCR routine total coliform/E. coli is generally presence/absence.
- RTCR sample volume is 100 mL.
- Chlorinated microbiological samples require chlorine-neutralizing preservative.
- Sample hold time and certified-lab requirements matter.
- Source-water monitoring and routine RTCR testing may use different logic.
- DBP, inorganic, secondary, organic, radionuclide, and UCMR method tables support method recognition.
