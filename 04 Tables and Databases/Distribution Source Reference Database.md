## Exam-Safe Summary Facts

| Concept | Exam-Safe Statement |
|---|---|
| Distribution systems | Deliver potable water while maintaining pressure, flow, and quality. |
| Looped systems | Improve circulation, redundancy, and operational flexibility. |
| Dead ends | Can create water-age and residual-loss issues. |
| Hydrants | Primarily installed for fire protection. |
| Dry-barrel hydrants | Intended for freezing climates. |
| Pipeline disinfection | Sanitary construction, flushing, disinfection, testing, and documentation matter. |
| Chlorine pH effect | Higher pH shifts free chlorine toward less-effective hypochlorite ion. |
| Membranes | RO/NF generally address dissolved material more than MF/UF. |
| Activated carbon | Used for taste/odor and some organic contaminant adsorption. |
